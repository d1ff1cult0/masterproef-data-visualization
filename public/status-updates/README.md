# Status Update Images

Place images for each status update in the corresponding subfolder:

- `update1/` – Status Update 1 (LEAR, DNN baseline results)
  - `lear_results_begin.png` – LEAR predictions at beginning of test set
  - `lear_results_einde.png` – LEAR predictions at end of test set
  - `dnn_results.png` – DNN predictions

- `update2/` – Status Update 2 (Price distribution, volatile period, model comparison)
  - `price_distribution_comparison.png` – Real distribution vs best 4 fits
  - `compare_datasets_metrics.png` – Metrics on old vs new dataset
  - `price_over_time_anomaly.png` – Price over time with volatile period marked
  - `time_series_period_comparison.png` – Time series for period 1 and 2
  - `price_distribution_comparison2.png` – Price distribution histogram for both periods

- `update3/` – Status Update 3 (Lag, output head, preprocessing)
  - `head_comparison.png` – Output head architecture comparison chart

- `update5/` – Status Update 5 (Notebook structure, EDA, price floor)
  - `prices_distribution.png` – Price distribution: histogram, box plot, Q-Q plot
  - `volatility_per_hour.png` – Volatility by hour of day

You can copy or export images from your LaTeX `latex/status_updates/images/` folder (convert PDF to PNG if needed).
